Sistema Bancario Santander Dev Week

O Sistema Bancário Santander foi um projeto que iniciei na Dev Week Santander com parceria com a DIO. Nessa semana elaborei o front-end com Angularjs e back-end foi criada uma api com Spring boot. As principais funções elaboradas nessa semana foram: visualizar extrato, adicionar saldo e remover valor. Logo Depois eu criei algumas features como: Login, Esqueci senha, autenticação de dois fatores, localização de acesso, recursos de liberar produtos por histórico ou perfil, minhas metas, empréstimo e We DIY.

Sistema Philips de Câncer de Mama

O Sistema Philips de Câncer de Mama foi um projeto que iniciei na Dev Week Philips com parceria com a DIO. O Projeto foi elaborado com front-end AngularJS e Backend Spring Boot e usamos uma api pública do SUS. Nessa semana as funções foram exibir as informações da API. Eu resolvi incrementar algumas funções como: Sistema de prontuário integrado a um banco de posts usando persistência, sistema de relatório, campo informação, criador de formulário, procedimento de consulta que alimentava mais a api.

Sistema de Gestão Financeiro Pessoal

Eu durante muito tempo tentei organizar minhas finanças com planilhas tanto local como no google drive, mas não foi bem sucedido. Então para resolver essa situação, depois de criar 3 sistemas com java e angular, resolvi criar o 4, que é um sistema de Gestão financeira Pessoal. Com base em um layout de uma planilha que encontrei na internet, criei um sistema com gráficos, relatórios, inserir cartão, fluxo de caixa e muito mais que eu precisava para ter o maior controle dos meus gastos e meus ganhos e terminar o mês com as finanças em dia.

Sistema de Monitoramento de Localizador

O Sistema de Monitoramento de Localizador foi um projeto que iniciei devido ao problema de acompanhar vários localizadores de passagens emitida por mim em planilha, app da companhia, bloco de notas ou até mesmo no e-mail. Desenvolvido com python na parte de raspagem de dados e criação de api, criação de api spring Boot para funções como: login easy, persistência no postgres, alerta de atualizações, integração email, criação de filtros de pesquisa e data e lembretes via canais. O front-end foi com angularjs.
